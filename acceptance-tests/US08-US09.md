## This acceptance test is for the following user stories and features

#### US.08 Scan a ticket to check validity
#### US.09 Export attendees list in CSV

## US.08 Scan a ticket to check validity
#### 1. Login with an approved Organizer account. You can login with the following info:
```
organizer id : O12345
password: 12345
```

#### 2. Open the Dashboard web page in your browser
  - Expected result:
  - Organizer dashboard loads with the organization’s name and an event selection dropdown.

#### 3. Select an event in the dropdown
  - Expected result:
    - A table with event stats and a table with guest list is displayed

#### 4. Click on ```Scan a ticket``` in the guest list header
  - Expected result:
    - A popup to upload an image to scan appears

#### 5. Click on the file upload button and select a QR code image that encodes a valid ticket number. You can use QR codes from the ```qr-codes``` folder.
  - ```VALID-tech-club-ai-ml.png``` is a valid ticket for "AI & Machine Learning Talk" by Tech Club
  - You may also login as a student, claim a ticket for an event and download the QR code for the image to then scan it
  - Expected result:
    - The system scans the uploaded image successfully.
    - The message area updates to: Ticket XXXX is VALID and displays a table with the guest information
  - The text is displayed in green color.

#### 6. Upload another QR code image that encodes an invalid ticket number. You can use QR codes from the ```qr-codes``` folder. 
  - Expected result: 
    - The system decodes the image. 
    - The message area updates to: Ticket XXXX is INVALID 
    - The text is displayed in red color.

#### 7. Upload a random image (no QR code). 
  - Expected result: 
    - The system tries to scan the image and then displays: 
    - Could not read QR code. Please try another image.
    - The text appears in red color.
    - No crash

## US.09 Export attendees list in CSV
#### 1. Login with an approved Organizer account. You can login with the following info:
```
organizer id : O12345
password: 12345
```

#### 2. Open the Dashboard web page in your browser
  - Expected result:
  - Organizer dashboard loads with the organization’s name and an event selection dropdown.

#### 3. Wrong User Type Step: Try to open the page as a student or guest. 
  - Expected Result: “No Access” message appears.

#### 4. View Events Step: Click the dropdown to see available events. 
  - Expected Result: All events created by the organizer appear in the list.

#### 5. Select an Event Step: Choose one event from the dropdown. 
  - Expected Result: Event details show up (date, capacity, tickets issued, remaining tickets).

#### 6. View Guest List Step: After selecting an event with guests. 
  - Expected Result: Guest table displays student ID, name, email, phone number, and ticket number. 
  - If there are no guests → message: “No guests.”

#### 7. Export Guest List Step: Click the “Export CSV” button. 
  - Expected Result: A CSV file with the guest list downloads automatically.

#### 8. Change Event Step: Select another event. 
  - Expected Result: Event and guest info update correctly without reloading the page.

#### 9. Past Event Attendance Step: Select an event that already happened. 
  - Expected Result: Checked-in and attendance rate data appear.
