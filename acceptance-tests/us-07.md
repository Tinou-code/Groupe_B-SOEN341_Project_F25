## This acceptance test is for the following user stories and features

#### US.07 As an organizer, I want to view a dashboard per event showing tickets issued, attendance rates, and remaining capacity, so I can monitor event engagement.


## Steps and expected results 
1. Login as an organizer. You can login with the following info:
Organizer id : O12345
password: 12345

2. Click on ```My Events``` in the sidebar - this will show you a list of events created by the organizer.

3. Click on any event from the list to open its details page.

4. Click on ```Dashboard``` to view analytics for that specific event.

5. The dashboard should display:
   - Tickets Issued
   - Attendance / Check-ins
   - Capacity
   - Remaining Capacity
   - Attendance Rate

#### Expected results
- The dashboard accurately displays event statistics for the selected event.
- Attendance Rate and Remaining Capacity are calculated correctly based on event data.
- The data updates when a different event is selected.
- Only the organizer who created the event can view its dashboard.
