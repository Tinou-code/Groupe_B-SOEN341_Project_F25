## This acceptance test is for the following user stories and features

#### US.10 As an administrator, I want to approve organizer accounts, to verify that organizers can create events


## Steps and expected results 
1. Login as a administrator. You can login with the following info:
```
Admin id : A12345
password: 12345
``` 
2. Click on ```Manage organizations``` in the sidebar - this will show you a list of all existing organizations. You can click on any existing organization and see their members.
3. You can see their account status and a button to either approve their account or disable their account. 
4. To approve or disable someone account click on the button.

#### Expected results
- If the organizer was not already approved, he will now be approved and will have access to create events.
- If the organizer was already approved, he will no longer have access to create events

- You can always change the status of an organizer account
