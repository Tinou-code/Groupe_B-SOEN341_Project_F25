## This acceptance test is for the following user story and feature

#### US.11 Create Admin view of list of events

## Steps and expected results 
1. Login as an admin. You can login with the following info:
```
Admin id : A12345
password: 12345
``` 
2. Click on ```Manage organizations``` in the sidebar - this will allow you to see all events depending on the organization you choose. Each event is displayed in a table, and you have the option as an admin to suspend or approve an event, which portraits the moderating nature of the admin role.

#### Expected results
- The event in question will be shown in list of events as approved or suspended depending on the action
- On screen notification confirms that the event was approved/suspended
- Green ```approve event``` button changes to red ```suspend event``` button

- If an error occurs, an on screen notification displays the error message. Errors shouldn't happen, but can be:
  - Not having the app launched correctly or it might be an internet issue
